The cool thing about computer science is that it uses information about the real world (called data) to make fun things with computers, like
games
calculators
books
and pretty much anything else you can think of!

For example, if you're going to make a game about a dog, the computer might need some data that comes from information about the real world. It might need to know the dog's name, age, favorite color, and maybe if it likes treats or not.

In the real world, we have many different types of things that we need to think about.

The computer knows about these different types of things so that we can represent lots of different real-world data.

Let's go over a few of the most common data types!

A string is a word, like "computer" or "hello" or "fun".
